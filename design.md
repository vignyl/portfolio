# Portfolio Design Style Guide

## Design Philosophy

### Color Palette
- **Primary**: Deep charcoal (#1a1a1a) and warm white (#fafafa)
- **Accent**: Soft amber (#f59e0b) for highlights and interactive elements
- **Secondary**: Muted teal (#0d9488) for secondary actions and data visualization
- **Neutral**: Cool gray (#64748b) for supporting text and subtle elements

### Typography
- **Display Font**: "Playfair Display" - elegant serif for headings and hero text
- **Body Font**: "Inter" - clean, modern sans-serif for readability
- **Code Font**: "JetBrains Mono" - for technical content and code snippets

### Visual Language
- Minimalist aesthetic with purposeful white space
- Subtle gradients and soft shadows for depth
- Geometric shapes and clean lines
- Professional yet approachable tone
- Focus on content hierarchy and readability

## Visual Effects & Animations

### Used Libraries
- **Anime.js**: Core animation engine for smooth transitions and micro-interactions
- **ECharts.js**: Interactive data visualizations for skills and project metrics
- **Typed.js**: Typewriter effects for hero section
- **Splitting.js**: Text animation effects for section reveals
- **Matter.js**: Physics-based animations for floating tech icons
- **Pixi.js**: Advanced visual effects for background elements

### Animation Style
- **Timing**: 300-600ms duration for most transitions
- **Easing**: Cubic-bezier curves for natural motion
- **Stagger**: 50-100ms delays between sequential elements
- **Parallax**: Subtle depth effects on scroll (max 8% translation)

### Header Effect
- Animated particle system background using Pixi.js
- Floating technology icons with physics-based movement
- Gradient text animation for name and title
- Smooth scroll indicator with bounce animation

### Interactive Elements
- 3D tilt effects on hover for cards and buttons
- Color morphing for navigation elements
- Shadow expansion for depth perception
- Smooth state transitions for form elements

### Background Treatment
- Consistent dark gradient background throughout
- Subtle geometric patterns using CSS transforms
- No section-specific background colors
- Maintained visual continuity across all pages

### Data Visualization Style
- Monochromatic color schemes for charts
- Maximum 3 colors per visualization
- Soft, rounded chart elements
- Interactive hover states with data tooltips
- Consistent spacing and typography

### Responsive Design
- Mobile-first approach with fluid typography
- Flexible grid system using CSS Grid and Flexbox
- Touch-friendly interactive elements (44px minimum)
- Optimized animations for reduced motion preferences
